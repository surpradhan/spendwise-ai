Features

Everything you need to understand your spending

No subscriptions, no data sharing, no SaaS lock-in — just a Python script and your data.

🤖

ML-Powered Categorisation

TF-IDF + logistic regression learns from your labelling history and fills in what keywords miss — getting smarter with every run.

📊

Interactive Dashboard

Seven Plotly charts — donut, monthly trend, top merchants, income vs expenses — in a single self-contained HTML file. Works offline.

🔁

Recurring Charge Detection

Automatically surfaces subscriptions and regular payments — no configuration needed. See frequency, median gap, and first/last dates.

💰

Budget Alerts

Set monthly limits per category with --set-budget "Groceries:400". Get warned at 80 % and notified when you exceed.

📄

PDF Reports

Multi-page PDF export for archiving or sharing. Same charts as the dashboard, rendered to static images via kaleido + reportlab.

⚙️

Pipe-Friendly JSON Mode

--json --no-feedback outputs a structured JSON summary — easy to pipe into other tools, scripts, or cron jobs.

How it works

From raw export to insight in one command

A deterministic, reproducible pipeline — every step is transparent and auditable.

📁

Ingest

Auto-detects encoding, normalises dates & amounts, masks card numbers

🏷️

Keyword Pass

Case-insensitive substring matching against your keywords.json

🤖

ML Pass

TF-IDF + logistic regression fills remaining uncategorised rows

📊

Summarise

Budget evaluation, recurring detection, terminal & JSON output

🚀

Export

Interactive HTML dashboard and optional PDF report

Quick Start

Up and running in < 10 minutes

Requires Python 3.10+. No accounts, no API keys.

# 1. Clone and install
git clone https://github.com/surpradhan/spendwise-ai
pip install -r requirements.txt
 
# 2. Drop your bank export into data/raw/, then run
python main.py --file data/raw/export.csv --dashboard
 
# With PDF report
python main.py --file data/raw/export.csv --dashboard --pdf
 
# Set budget limits, then run
python main.py --file data/raw/export.csv \
--set-budget "Groceries:400" "Transport:100" --dashboard
 
# Retrain ML model from your labelled history
python main.py --file data/raw/export.csv --retrain-ml

Supported formats: CSV, XLSX, XLS  ·  Required columns: Date, Description, Amount (you'll be prompted to map them if the names differ)

Privacy

Your data never leaves your machine

SpendWise AI is built on a simple principle: your financial data is yours.

🔒

100% Local Processing

No internet connection required after install. All analysis runs on your hardware.

🎭

Automatic PII Masking

Card numbers (12–16 digits) are masked to ****1234 in all outputs — terminal, CSV, and dashboard.

📡

Zero Telemetry

No analytics, no crash reports, no logging to external services. Ever.

🔍

Open Source

Every line of code is auditable. MIT licensed — use it, fork it, modify it.
